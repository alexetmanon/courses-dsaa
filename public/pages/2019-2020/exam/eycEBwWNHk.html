Objectif

Vous devez intégrer la page comme présentée dans l'encadré suivant.
Quelques consignes importantes :

  • La police utilisée devra être Arial sur l'ensemble de la page (hors bonus)
  • Le texte voir le profil devra être un lien cliquable qui mène vers le site de votre choix
  • Le contenu de la section ACTIVITÉ LIÉE À est une liste à puce à deux éléments
  • La carte sera remplacée par une image fournie (voir un plus bas)
L'ensemble des éléments pourront être positionnés en utilisant flexbox.

Il est possible que vous ayez de la difficulté à effectuer l'intégralité de l'exercice dans le temps imparti. Ne paniquez pas, ne vous perdez pas dans des détails et faite le maximum.

Règles

Pour ce faire vous aurez droit à :

  • Internet
  • Vos notes prises lors du cours

Vous n'avez pas le droit à :

  • Vos précédent fichiers produits en cours (ou ailleurs)
  • L'appel à un ami (de quel que forme soit-il)

Format de rendu

Vous m'enverrez une archive zip de votre dossier de code contenant l'ensemble de vos fichiers (html, css, font, image, etc.).
Je dois pouvoir afficher votre site sur ma machine sans autre document que celui-ci.
